# 概要

バニラのかまどに速度選択機能が追加されます。
7段階の速度から選択可能です：1x（デフォルト）、5x、10x、15x、20x、25x、30x。
既存のかまどで動作します。

## 操作方法

1. かまどを開く。
2. 「速度: Nx」をクリックして切り替え:
   - `1x → 5x → 10x → 15x → 20x → 25x → 30x → 再びループ。`

## 必要条件

- **Minecraft**: 1.21.10
- **Java**: 21
- **Fabric Loader**: 0.17.3+
- **Fabric API**: 0.136.0+1.21.10
