# 概要

バニラのかまどに倍率操作ボタンを追加します。
6段階の速度が利用可能です: 1x (デフォルト), 10x, 20x, 30x, 40x, 50x。
従来のかまどでそのまま利用できます。

## 操作方法

1. かまどを開く。
2. 「速度: Nx」ボタンをクリックして切り替える。
   - 1x → 10x → 20x → 30x → 40x → 50x → ループ。

## 必要環境

- **Minecraft**: 1.21.10
- **Java**: 21
- **Fabric Loader**: 0.17.3+
- **Fabric API**: 0.136.0+1.21.10
